# AI Writing Instructions

## Languages
Supported languages:
- German (DE)
- English (EN)
- French (FR)
- Spanish (ES)

Language selection must be based on:
- festival country
- available band materials
- explicit configuration

## Content Rules
- Use only provided band materials
- Never invent achievements, awards, or contacts
- Tone depends on festival size

## Tone
- Small festivals (<2000): friendly, concise
- Large festivals (>5000): formal, structured

## Mandatory Elements
Every application must include:
- festival name
- festival dates
- festival location
- band name
